Projetos

Aplicação de Monitoramento de Dados Ambientais
2022 - Projeto Acadêmico
Criação de um sistema para coleta e visualização de dados ambientais, como temperatura, qualidade do ar e umidade. A aplicação integra sensores simulados e apresenta os dados em dashboards interativos para análise em tempo real.
Tecnologias utilizadas: Python, Flask, PostgreSQL e bibliotecas de visualização de dados.

Plataforma Web para Gestão de Doações
2023 - Projeto Pessoal
Desenvolvimento de uma plataforma web voltada para conectar doadores a instituições de caridade. O sistema permite cadastro de organizações, gerenciamento de campanhas e acompanhamento de doações em tempo real.
Tecnologias utilizadas: JavaScript, React, Node.js, MongoDB e API REST.
